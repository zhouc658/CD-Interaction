Html Exploration

This assignment is about creating a series of 5 unique web pages which focus on one or two HTML tags each to see how html tags look by defualt and how we can use them in an inventive way. I decided to focus on using headings, paragraphs, images, listing, and navigation.

Note: The heading tags are also used on other pages to help organize content more clearly. Likewise, the navigation tag appears on other pages to allow users to navigate between them easily.
